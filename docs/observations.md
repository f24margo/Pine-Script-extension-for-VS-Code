## Март 2025: Резкое падение (Crash)
- **Дата:** 01.03.2025
- **Цена:** 2020.41 (Зона Discount 1.9%)
- **VIX:** 23.37 (Зона WARNING)
- **Заметка:** Наблюдается сильный Bearish Bias. При таком VIX стандартные стоп-лоссы становятся уязвимыми. Индикатор APEX может показывать расширение диапазонов. Необходимо следить за удержанием уровня 2000.00.
Интересное наблюдение — смотри что видно.

**Три линии MTF на H4 графике:**
```
Жёлтая  → 3D Score   — медленная, плавная
Голубая → Daily Score — активная, быстрая
Оранжевая → H4 Score  — самая активная
```

**Что читается прямо сейчас (5 марта):**
```
3D Score    → 80%  — три таймфрейма подтверждают
Daily Score → 80%  — совпадает
H4 Score    → 60%  — немного слабее
Зелёный фон → есть — все три выше порога 3
```

**Ключевое наблюдение на периоде 13 фев — 5 мар:**

```
13-17 фев:  3D падает → Daily падает следом через 1-2 бара
            Старший ТФ предупреждает раньше

22-23 фев:  Все три выровнялись вверх → зелёный фон
            На графике в этот момент — разворот вверх

25-27 фев:  3D держится высоко, Daily падает
            Сигнал: старший ТФ не подтверждает падение
            → откат а не разворот

Март 1-5:   Все три растут одновременно
            Зелёный фон расширяется
```

**Главный вывод МТФ:**

3D линия работает как фильтр — когда она падает а Daily высоко, это ложный сигнал. Когда обе растут одновременно — сигнал надёжнее.
