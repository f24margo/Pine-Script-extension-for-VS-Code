Этот раздел предназначен для торговых алгоритмов, использующих функции `strategy.*` для тестирования на исторических данных и автоматизации торговли.

## Основные отличия от индикаторов:
1. **Заголовок**: Используется `strategy()` вместо `indicator()`.
2. **Исполнение**: Возможность открытия сделок через `strategy.entry()` и закрытия через `strategy.exit()`.
3. **Тестирование**: Доступна вкладка "Тестер стратегий" (Strategy Tester) в TradingView.
